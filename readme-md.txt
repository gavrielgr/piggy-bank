# קופת החיסכון המשפחתית 💰

אפליקציית קופת חיסכון דיגיטלית לילדים, שנועדה לעזור להם ללמוד על ניהול כספים, חיסכון וריבית בצורה מהנה וחווייתית.

## תכונות עיקריות ✨

- **ניהול חסכונות משפחתי** - לכל ילד יש קופת חיסכון עם ניהול עצמאי
- **מערכת ריבית** - סימולציה של ריבית חודשית לעידוד חיסכון ארוך-טווח
- **הגדרת מטרות** - יצירת מטרות חיסכון אישיות עם מעקב התקדמות
- **משחוק (Gamification)** - רמות, הישגים ותגים להגברת המוטיבציה
- **ויזואליזציות יצירתיות** - עץ שגדל או חללית שמתקרבת לכוכב ככל שהחיסכון גדל
- **סימולטור החלטות פיננסיות** - למידה על תוצאות של החלטות כלכליות
- **תמיכה במובייל** - ניתן להתקין כאפליקציה על מכשירים ניידים (PWA)
- **אזור מיוחד להורים** - בקרת הורים עם סיסמה, הגדרות וסינכרון נתונים

## התקנה 🔧

### הורדה מ-GitHub

```bash
git clone https://github.com/username/piggy-bank.git
cd piggy-bank
```

### הפעלה מקומית

אין צורך בהתקנת ספריות או כלים - זהו פרויקט שפועל במלואו בצד הלקוח.
ניתן להשתמש בכל שרת סטטי:

```bash
# עם Python
python -m http.server

# או עם Node.js
npx serve
```

## מבנה הפרויקט 📁

```
piggy-bank/
├── index.html               # דף ראשי
├── manifest.json            # הגדרות PWA
├── favicon.ico              # אייקון האתר
├── assets/                  # תמונות ואייקונים
├── css/                     # קבצי CSS
│   ├── style.css            # סגנונות כלליים
│   ├── dark-mode.css        # סגנונות למצב כהה
│   └── responsive.css       # סגנונות רספונסיביים
└── js/                      # קבצי JavaScript
    ├── app.js               # קוד ראשי
    ├── state.js             # ניהול מצב האפליקציה
    ├── ui.js                # עדכוני ממשק משתמש
    ├── charts.js            # גרפים וויזואליזציות
    ├── goals.js             # ניהול מטרות
    ├── gamification.js      # מערכת משחוק והישגים
    ├── transactions.js      # ניהול עסקאות
    ├── simulator.js         # סימולטור החלטות
    ├── parent-dashboard.js  # פאנל בקרת הורים
    └── utils.js             # פונקציות עזר
```

## טכנולוגיות 🛠️

- **HTML5** - מבנה האפליקציה ותמיכה במאפייני מובייל
- **CSS3** - עיצוב מודרני עם מצב כהה ותמיכה במסכי מגע
- **JavaScript** - לוגיקה ואינטראקציה
- **LocalStorage** - אחסון נתונים מקומי
- **PWA** - תמיכה בהתקנה כאפליקציה

## הערות לפיתוח 💻

- האפליקציה פועלת במלואה בצד הלקוח ללא צורך בשרת
- נתונים נשמרים מקומית במכשיר המשתמש באמצעות LocalStorage
- ניתן ליצור גיבויים ולשחזר נתונים דרך פאנל ההורים
